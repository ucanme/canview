# 🎨 Zed风格设计文档

## 🌟 设计理念

CanView现在采用了与Zed编辑器相同的现代化设计语言，提供简洁、优雅且功能强大的用户界面。

## 🎯 设计特色

### 配色方案
遵循Zed的深色主题配色：
- **主背景**: `#181818` - 深沉稳重的主色调
- **次级背景**: `#1f1f1f` - 稍亮的容器背景
- **边框颜色**: `#2a2a2a` - 微妙的分界线
- **主文字**: `#ffffff` - 纯白标题和重要信息
- **次级文字**: `#d1d5db` - 灰白色常规内容
- **辅助文字**: `#9ca3af` - 淡灰色提示信息
- **暗淡文字**: `#6b7280` - 最淡的状态信息

### 语法高亮色彩
借鉴代码编辑器的语法高亮：
- **时间戳**: `#9ca3af` - 淡灰色，低调显示
- **通道号**: `#60a5fa` - 蓝色，表示连接信息
- **消息类型**: `#34d399` - 绿色，表示协议类型
- **消息ID**: `#fbbf24` - 黄色，突出显示标识符
- **数据内容**: `#a78bfa` - 紫色，表示数据内容
- **信号解析**: `#9ca3af` - 淡灰色，辅助信息

### 交互反馈
- **悬停效果**: `#1f2937` - 微妙的高亮反馈
- **选中状态**: `#3b82f6` - Zed经典蓝色强调
- **活动标签**: 蓝色底边线指示当前页面

## 🏗️ 界面布局

### 1. 标题栏 (40px高度)
```
┌─────────────────────────────────────────────────────────────┐
│ CanView / logs          status message              Open Config │
└─────────────────────────────────────────────────────────────┘
```

**特点**:
- 极简设计，只显示核心信息
- 左侧：应用名称 + 面包屑导航
- 中央：状态信息显示
- 右侧：快速操作按钮

### 2. 标签栏 (36px高度)
```
┌─────────────────────────────────────────────────────────────┐
│ [Logs] Configuration Analytics        123 messages 2DBC 1LIN │
└─────────────────────────────────────────────────────────────┘
```

**特点**:
- Zed风格的标签设计
- 活动标签有蓝色底边线
- 右侧显示实时统计信息
- 悬停效果增强交互体验

### 3. 内容区域
**Logs视图**:
- 深色表格，交替行背景
- 语法高亮的数据显示
- 悬停行高亮效果

**配置视图**:
- 卡片式布局

**分析视图**:

## 🎨 视觉元素

### 按钮样式
- **主按钮**: 深色背景 `#2a2a2a`，悬停时变亮 `#374151`
- **圆角**: 4px 统一圆角半径
- **字体**: 小号字体 (12px) 保持简洁

### 表格样式
- **表头**: 深色背景 `#1f1f1f`，全大写字母
- **数据行**: 交替背景色，悬停高亮
- **边框**: 极细边框 `#2a2a2a`

### 标签页样式
- **活动标签**: 白色文字，蓝色底边线
- **非活动标签**: 灰色文字，悬停变亮
- **顶部圆角**: 6px 营造现代感

## 🚀 技术实现

### 核心技术栈
- **GPUI 0.2.2**: GPU加速渲染引擎
- **原生组件**: 移除外部UI库，使用纯GPUI实现
- **响应式设计**: 基于flexbox的布局系统

### 性能特性
- **GPU渲染**: 硬件加速的流畅体验
- **内存优化**: 高效的数据结构
- **即时响应**: 零延迟的UI更新
